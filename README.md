# Financial Risk Management and Strategies for Crypto Assets

## Overview
This project is an AI-powered cryptocurrency analysis platform that predicts prices,
evaluates financial risk, and visualizes market trends using machine learning.

## Features
- Crypto price prediction using Random Forest
- Risk assessment (Volatility, Sharpe Ratio, VaR, Max Drawdown)
- Technical indicators (SMA5, SMA10, RSI)
- Interactive Streamlit dashboard
- Market sentiment and dominance analysis

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Plotly
- Pandas, NumPy
